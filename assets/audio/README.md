# Afar Audio Directory

Place your compressed Afar audio files here with the following naming convention:

- 001.ogg
- 002.ogg
- 003.ogg
- ...
- 604.ogg

Total: 604 audio files in OGG format (10-16 kbps compressed)

Each audio file corresponds to the Quran page with the same number.
For example: 001.ogg is the audio recitation for page 1.
